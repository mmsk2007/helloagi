# HelloAGI vision

HelloAGI is not another thin wrapper around a chat model. It is an **SRG-governed autonomous runtime**: deterministic policy gates on inputs, tools, memory, and outputs; a **learnable skill bank** fed by successful plans; **VLAA-style reliability** that refuses phantom completions and breaks loops; and **structured context** that respects token budgets instead of dumping history.

We compete on **safety you can audit** (journal + governance records), **skills that survive across sessions**, and **optional governed browser automation** aligned with the same policy packs as shell and file tools.

Non-goals: bypassing SRG for convenience, silent policy changes, or undeclared side effects.
